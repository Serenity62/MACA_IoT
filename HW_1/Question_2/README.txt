Homework 1 Question 2

Introduction

In Homework 1 Question 2 there are two parts. Part one creates a neural network using the pixel info method, while part 2 creates it using the frequency coefficients method.

Requirements

This project requires access to Matlab 2017a along with access to fer2013.csv which is the dataset for the project.

Usage

To use this project you must first store the dataset in the csv file on your computer, then run the matlab code provided.
